# 六、Crux实现

Crux用于 **DNN job scheduling**

核心：

```
GPU intensity
```

---

## Crux调度器

```python
class CruxScheduler(Scheduler):

    def compute_gpu_intensity(self, job):

        return job.compute_time / job.communication_time


    def assign_priorities(self, jobs):

        jobs.sort(key=lambda j: j.gpu_intensity, reverse=True)

        for i, job in enumerate(jobs):

            job.priority = i


    def compute_schedule(self, network_state):

        self.assign_priorities(network_state.jobs)

        for job in network_state.jobs:

            job.path = self.select_path(job)
```

    for f in active_flows:
